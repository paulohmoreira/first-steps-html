O que são Clients?

← Voltar

Trazendo para o bom e velho português, é o lado do cliente, onde o usuário (cliente) consegue visualizar, fazer requisições e receber respostas. Os navegadores (browsers) são um exemplo de software que permite o usuário visualizar páginas web, sendo assim podemos dizer que é o lado client da arquitetura web

  • Inicia solicitações para os servidores
  • Recebe respostas
  • Interage diretamente com os servidores

O que são Servers?

É o lado responsavél por lidar com as requisições do client, os servers ficam a espera dessas requisições, fazem o tratamento e devolvem uma resposta para o client

Como é uma estrutura básica de um HTML?

← Voltar

HTML é uma linguagem de marcação, ou seja, as tags HTML são marcadores que indicam para o navegador qual comportamento o texto dentro daquela tag deve ter.

Para o navegador entender um documento HTML, ele deve ter uma estrutura básica:

  1. <!DOCTYPE html> É a tag que define que este documento se trata de um documento HTML5
  2. <html> Está é a base de todo o documento, é dentro dela que colocaremos todo os elementos do nosso site
  3. <head> As informações contidas dentro da head são para passar informações diretamente para o navegador, informações que podem mudar a apresentação da página html
  4. <title> Aqui colocamos o título da página que aparece na aba do navegador
  5. <body> Corpo da página, aqui fica todos os conteúdos visíveis para o usuário

Linguagem de programção: o que é?

← Voltar

É um conjunto de instruções que o programador escreve e através disso o computador executa uma tarefa de acordo com as instruções que foram fornecidas. As linguagens podem ser compiladas, elas convertem o código para a linguagem de máquina, ou interpretadas, um interpretador fica responsável por ler o código e executar
